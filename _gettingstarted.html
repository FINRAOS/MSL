Getting Started

Starting Web Server

No matter what binding you have decided on, you will need to first get the WebServer. To do this, you will need to have nodejs already installed

npm install web-server

Choosing Client

Next, you will have to install the bindings that will be used to set up the requests to expect and the responses required for your application to run.

JavaScript Client

You can use the JavaScript client if you want to execute your JS tests on a browser.

NodeJS Client

You can use the NodeJS client if you want to execute your JS tests on NodeJS.

Java Client

You can use the Java client if you want to execute your tests using Java testing frameworks such as JUnit.
